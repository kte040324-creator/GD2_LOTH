# Web (static landing)

This folder contains a **static 1920px landing page** generated from Figma.

## Preview

- Open `apps/web/public/index.html` directly in a browser, **or**
- Run a simple static server:

```bash
cd apps/web/public
python3 -m http.server 5173
```

Then open `http://localhost:5173/`.

## Notes

- The design uses the font **Neue Haas Grotesk Display Pro**. If it is not installed on your machine, the browser will fall back to system fonts.
